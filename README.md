# OSC-YTmode-Display
 Use STM32F1 DAC Disp A graph in OSC Y-T mode


200 x 150 dots

8M HSE

PA4 - DAC Output
PA9 PA10 - USART

Usart Bandwidth: 1500000

AA + Graph Data(200 * 150 / 8 Bytes) + 00 00 00 00(for Clear and Disp)
